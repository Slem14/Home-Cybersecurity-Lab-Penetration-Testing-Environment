# Lab Architecture

The lab consists of two virtual machines:

- Kali Linux (Attacker)
- Metasploitable 2 (Victim)

Both machines are connected using a Host-Only Network in VirtualBox.

This ensures:
- Safe environment
- No external exposure
- Controlled penetration testing
