# Home Cybersecurity Lab – Penetration Testing

## Overview
This project demonstrates a penetration testing lab built using virtual machines.  
The goal is to simulate real-world cyber attacks and understand vulnerabilities, exploitation, and system compromise.

## Environment
- Kali Linux (Attacker)
- Metasploitable 2 (Target)
- VirtualBox (Virtualization)

## Lab Network
- Kali: 192.168.56.102
- Target: 192.168.56.101

## Tools Used
- Nmap
- Metasploit

## Key Achievements
- Performed network scanning and service enumeration
- Identified vulnerable FTP service (vsftpd 2.3.4)
- Exploited the service using Metasploit
- Gained root access to the target system
- Retrieved sensitive data (password hashes)
