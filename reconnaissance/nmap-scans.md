# Nmap Scan

## Command
nmap -sS -A 192.168.56.101

## Purpose
To identify open ports, services, and potential vulnerabilities.

## Key Findings
- FTP (vsftpd 2.3.4)
- SSH
- Telnet
- HTTP (Apache)
- Samba
- MySQL
- PostgreSQL
- Tomcat

## Analysis
The system exposes multiple services, increasing the attack surface and likelihood of compromise.
